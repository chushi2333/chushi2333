### 主要项目

#### AI Interview Assistant

一个面向程序员面试刷题场景的学习平台，后端基于 Spring Boot 构建，正在逐步加入 AI 助教能力。

目前已经完成的 AI 方向能力：

- AI 题目追问：围绕当前题目进行解释、提示和扩展回答
- 对话记忆：保存用户消息和模型回复，支持最近上下文拼接
- 会话摘要：将较长历史对话压缩成长期摘要，降低上下文成本
- 用户长期记忆：根据会话摘要沉淀用户学习画像
- RAG 检索：使用 PostgreSQL + pgvector 存储题目向量，支持相似度召回
- 多段 chunk：将题目内容和答案拆分成多个片段，提高检索粒度
- RAG 决策：通过规则判断是否需要触发 embedding 检索，减少无效模型调用
- 批量索引与定时索引：支持手动批量索引和定时索引最近题目
