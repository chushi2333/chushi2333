### 最近在做

**AI Interview Assistant** 是一个面向程序员面试刷题场景的学习平台。

我在这个项目里主要做后端和 AI 能力接入：

- 基于 Spring Boot 实现题库、用户、对话等基础业务接口
- 接入 LangChain4j 和 OpenAI-compatible 模型，完成 AI 题目追问
- 设计对话记忆、会话摘要和用户长期记忆，控制上下文长度
- 使用 PostgreSQL + pgvector 实现题目 RAG 检索
- 实现题目 chunk、embedding 写入、topK 召回和 distance 过滤
- 抽象 RAG 决策服务，减少无效 embedding 调用
- 增加批量索引和定时索引，方便维护向量库数据

### 当前重点

- Java 基础和常见集合源码
- Spring Boot 项目工程化
- MySQL 索引、事务和 SQL 优化
- Redis 基础使用场景
- AI 应用中的 Prompt、Memory、Embedding、RAG
- 项目复盘和面试表达
